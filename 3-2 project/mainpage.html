ABOUT

This project helps the students to track the location of the college bus easily by using the GPS . This project solves the real time problem, which students face in these days.One should enroll themselves , The students and faculty information i.e name and bus no. is stored in the data base ,if the students successfully enrolled they can know the bus location where they are exactly according to the route.This can help the student take another bus in the same route in case if they miss their bus by tracking the busses where exactly they are and even to track their own bus so that they can reach the bus .Vehicle tracking system using Google maps, GPS module used for getting location . Initially the GPS continuously takes input data from the satellite and stores the latitude and longitude values. If we have to track the vehicle, we need to send a message to GSM device, by which it gets activated. Wifi module to send data to mobile or computer over wifi.
